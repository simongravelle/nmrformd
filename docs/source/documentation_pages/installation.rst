============
Installation
============

The development version of NMRforMD can be compiled from source.
`NumPy`_ is required:

.. code-block:: bash

    pip3 install numpy

Then type in a terminal:

.. code-block:: bash

    git clone git@github.com:simongravelle/nmrformd.git
    cd nmrformd/
    pip3 install .

.. _`NumPy` : https://numpy.org/
